maps

Frequency count

Count occurrences of a key. Useful for histograms, mode-finding, and anagram checks.

Map<String, Integer> freq = new HashMap<>();
freq.merge(key, 1, Integer::sum);

Get or default + put

Retrieve a value with a fallback if the key is absent.

map.getOrDefault(key, 0) + 1

Compute if absent

Group items by key into lists or sets. Creates the collection on first access.

map.computeIfAbsent(key, k -> new ArrayList<>()).add(val);

Iterate entries

Loop over all key-value pairs in a map.

for (var e : map.entrySet()) {
    e.getKey(); e.getValue();
}

Max entry by value

Find the entry with the highest value in a map.

map.entrySet().stream()
    .max(Comparator.comparingInt(Map.Entry::getValue))
    .orElseThrow();

Remove entries matching predicate

Prune map values in-place. Here, keeping only keys whose list has duplicates.

map.values().removeIf(list -> list.size() < 2);

streams

Filter + collect

Keep elements matching a condition and collect into a list.

List<Integer> positive = list.stream()
    .filter(x -> x > 0)
    .collect(Collectors.toList());

Map + collect

Transform each element and collect results.

List<String> upper = list.stream()
    .map(String::toUpperCase)
    .collect(Collectors.toList());

Group by

Partition elements into a map of key to list. Here, grouping integers by even/odd.

Map<Integer, List<Integer>> groups = list.stream()
    .collect(Collectors.groupingBy(x -> x % 2));

FlatMap

Flatten nested collections into a single stream.

List<String> flat = nested.stream()
    .flatMap(Collection::stream)
    .collect(Collectors.toList());

IntStream range loop

Iterate over a range of integers without a traditional for-loop.

IntStream.range(0, n).forEach(i -> { /* use i */ });

Max / Sum / Average

Common numeric reductions on a list of integers.

int max = list.stream().mapToInt(Integer::intValue).max().orElse(0);
int sum = list.stream().mapToInt(Integer::intValue).sum();
double avg = list.stream().mapToInt(Integer::intValue).average().orElse(0.0);

Max by field on objects

Find the element with the highest value for a given property.

Person oldest = people.stream()
    .max(Comparator.comparingInt(Person::age))
    .orElseThrow();

Filter to new list

Produces a mutable ArrayList.

List<String> longNames = names.stream()
    .filter(n -> n.length() > 3)
    .collect(Collectors.toList());

Collect to Set / Map

Collect stream results into a Set or a Map keyed by a property.

Set<String> unique = list.stream().collect(Collectors.toSet());

Map<String, Integer> byName = people.stream()
    .collect(Collectors.toMap(Person::name, Person::age));

Joining strings

Concatenate stream elements with a delimiter.

String csv = list.stream().collect(Collectors.joining(", "));

concurrency

Thread with lambda

Start a new thread with inline logic.

Thread t = new Thread(() -> {
    // work
});
t.start();

Blocking queue (producer/consumer)

Thread-safe hand-off between producer and consumer threads. Both operations block when the queue is at capacity or empty.

BlockingQueue<T> q = new LinkedBlockingQueue<>();
q.put(item);   // blocks if full
q.take();      // blocks until available

Executor + join all

Submit tasks to a thread pool and wait for all to complete.

ExecutorService exec = Executors.newFixedThreadPool(n);
exec.submit(() -> { /* task */ });
exec.shutdown();
exec.awaitTermination(Long.MAX_VALUE, TimeUnit.MILLISECONDS);

try-with-resources on ExecutorService (Java 19+)

Auto-shutdown when the block exits.

try (var executor = Executors.newFixedThreadPool(n)) {
    executor.submit(() -> { /* task */ });
}  // auto shutdown

Atomic result from thread

Return a value from a thread when no Future is available.

AtomicInteger result = new AtomicInteger();
Thread t = new Thread(() -> result.set(42));
t.start();
t.join();
int value = result.get();

} catch (InterruptedException e) {

Thread + join all

When you need per-worker identity (e.g. mailboxes, partitions).

List<Thread> threads = new ArrayList<>();
for (int i = 0; i < n; i++) {
    final int id = i;
    Thread t = new Thread(() -> { /* use id */ });
    t.start();
    threads.add(t);
}
for (Thread t : threads) t.join();

Per-key rate limiting

Throttle per host/key without blocking other keys. Computes the wait time inside the atomic compute call, then sleeps outside the lock.

ConcurrentHashMap<String, Long> lastAccessTime = new ConcurrentHashMap<>();

long[] wait = {0};
lastAccessTime.compute(host, (h, lastTime) -> {
    long now = System.currentTimeMillis();
    if (lastTime != null) {
        long elapsed = now - lastTime;
        if (elapsed < delayMs) {
            wait[0] = delayMs - elapsed;
            return now + wait[0];
        }
    }
    return now;
});
if (wait[0] > 0) Thread.sleep(wait[0]); // sleep OUTSIDE compute

ForkJoinPool for custom parallelism

Bound parallelStream to N threads instead of using the common pool.

ForkJoinPool pool = new ForkJoinPool(numThreads);
try {
    pool.submit(() ->
        files.parallelStream().forEach(path -> { /* process */ })
    ).get();
} catch (ExecutionException e) {
    throw new RuntimeException(e.getCause());
} finally {
    pool.shutdown();
}

Batch futures (round-based BFS)

Concurrent crawl/search where each round discovers the next frontier. Submit all work, then collect results before the next round.

Queue<String> toProcess = new LinkedList<>();
Set<String> visited = new HashSet<>();
toProcess.add(seed);

while (!toProcess.isEmpty()) {
    List<Future<List<String>>> futures = new ArrayList<>();
    while (!toProcess.isEmpty())
        futures.add(executor.submit(() -> fetcher.fetch(toProcess.poll())));
    for (var f : futures) {
        for (String result : f.get()) {
            if (visited.add(result)) toProcess.add(result);
        }
    }
}

Stream chunk hash

Hash large files without loading the entire contents into memory.

var md = MessageDigest.getInstance("SHA-256");
try (var is = Files.newInputStream(path)) {
    byte[] buf = new byte[8192];
    int n;
    while ((n = is.read(buf)) != -1)
        md.update(buf, 0, n);
}
byte[] hash = md.digest();

patterns

BFS skeleton

Breadth-first search over a graph or state space.

Queue<T> q = new LinkedList<>();
Set<T> visited = new HashSet<>();
q.add(start);
visited.add(start);

while (!q.isEmpty()) {
    T cur = q.poll();
    for (T next : neighbors(cur))
        if (visited.add(next)) q.add(next);
}

BFS reachability check

Determine if a target node is reachable from a source in a directed graph.

boolean isReachable(String from, String target,
                     Map<String, Set<String>> graph) {
    var visited = new HashSet<String>();
    var queue = new LinkedList<String>();
    queue.add(from);
    while (!queue.isEmpty()) {
        var current = queue.poll();
        if (current.equals(target)) return true;
        if (visited.add(current))
            queue.addAll(graph.getOrDefault(current, Set.of()));
    }
    return false;
}

DFS skeleton (iterative)

Depth-first search using an explicit stack. Avoids recursion stack limits on deep graphs.

Deque<T> stack = new ArrayDeque<>();
Set<T> visited = new HashSet<>();
stack.push(start);
visited.add(start);

while (!stack.isEmpty()) {
    T cur = stack.pop();
    for (T next : neighbors(cur))
        if (visited.add(next)) stack.push(next);
}

DFS skeleton (recursive)

Cleaner when you need pre/post-order processing or backtracking.

Set<T> visited = new HashSet<>();

void dfs(T node) {
    visited.add(node);
    // pre-order work here
    for (T next : neighbors(node))
        if (!visited.contains(next)) dfs(next);
    // post-order work here
}

Prefix match length

Find where two sequences diverge. Useful for comparing stack traces or computing diffs.

int prefix = 0;
while (prefix < a.size() && prefix < b.size()
       && a.get(prefix).equals(b.get(prefix)))
    prefix++;

Poison pill shutdown

Signal consumers to stop by placing a sentinel value on the queue. Each consumer re-posts the poison so all workers shut down.

static final String POISON = "__STOP__";

// producer
queue.put(POISON);

// consumer
String item = queue.take();
if (item.equals(POISON)) {
    queue.put(POISON);  // re-post for other consumers
    return;
}

LRU Cache (LinkedHashMap)

Pass true for access-order mode. Override removeEldestEntry for automatic eviction when the cache exceeds capacity.

int capacity = 100;
LinkedHashMap<K, V> cache = new LinkedHashMap<>(capacity + 1, 0.75f, true) {
    @Override
    protected boolean removeEldestEntry(Map.Entry<K, V> eldest) {
        return size() > capacity;
    }
};
// get() promotes to most-recent; iteration = least → most recent

Trie insert + longest match

O(L) vocabulary lookup for tokenizers, autocomplete, or IP routing tables.

// insert a word
TrieNode node = root;
for (char c : word.toCharArray())
    node = node.children.computeIfAbsent(c, k -> new TrieNode());
node.isEnd = true;

// find longest match starting at index i
TrieNode n = root;
int matchLen = 0;
for (int j = i; j < text.length(); j++) {
    n = n.children.get(text.charAt(j));
    if (n == null) break;
    if (n.isEnd) matchLen = j - i + 1;
}

Multi-stage dedup

Progressively expensive checks to find duplicate files. Group by size first (cheap), then partial hash, then full hash. Non-duplicates are pruned early.

// Stage 1: group by file size (cheap)
Map<Long, List<Path>> bySize = new HashMap<>();
Files.walk(dir).filter(Files::isRegularFile)
    .forEach(p -> bySize.computeIfAbsent(
        Files.size(p), k -> new ArrayList<>()).add(p));
bySize.values().removeIf(l -> l.size() < 2);

// Stage 2: partial hash (first 4KB) on size-matched groups
// Stage 3: full hash on partial-hash-matched groups

persistence / wal

Write-ahead log (append mode)

Log mutations before applying them so they can be replayed after a crash.

BufferedWriter wal = Files.newBufferedWriter(walPath,
    StandardOpenOption.CREATE, StandardOpenOption.APPEND);

// on each put:
wal.write("PUT\t" + key + "\t" + value + "\n");
wal.flush();  // flush immediately for durability

WAL recovery (replay)

Re-apply logged mutations on startup. Malformed lines from mid-write crashes are skipped.

for (String line : Files.readAllLines(walPath)) {
    String[] parts = line.split("\t", 3);
    if (parts.length == 3 && "PUT".equals(parts[0]))
        map.put(parts[1], parts[2]);
    // skip malformed lines (partial writes from crash)
}

Escape/unescape for delimited formats

When values might contain your delimiter (e.g. tabs).

static String escape(String s) {
    return s.replace("\\", "\\\\")
            .replace("\t", "\\t")
            .replace("\n", "\\n");
}

Collision-free cache key

Build a cache key from mixed types or variable-length parts without ambiguity. Uses type and length prefixes to prevent collisions.

StringBuilder sb = new StringBuilder();
for (Object part : parts) {
    String type = part.getClass().getSimpleName();
    String val = String.valueOf(part);
    sb.append(type).append(':').append(val.length())
      .append(':').append(val).append('\u001f');
}
String cacheKey = sb.toString();
